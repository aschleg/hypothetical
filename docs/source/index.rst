

Welcome to hypothetical's documentation!
----------------------------------------

Hypothetical is a python library for conducting hypothesis, statistical and other group comparison tests.

Installation
============

.. code-block:: console

    pip install hypothetical

For those interested, the most recent development version of the library can also be installed by cloning or
downloading the repo.

.. code-block:: console

    git clone git@github.com:aschleg/hypothetical.git
    cd hypothetical
    python setup.py install
